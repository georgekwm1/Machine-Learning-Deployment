How It Works

1

Before uploading a file, the data should be cleaned. The cleaning involves removing any column that has rows which contains any alphabet or words as values. The column names should be at the first row of the file. If any column contains a word or alphabet, the column will be completely removed and not used for analysis. The file should only contain Numerical values. The column of what you intend to predict must be placed as the last column in the file.

2

Select and Upload a file (csv or excel file). This is the file that will be used to train the model that will be predicted. Give it a little time to analyze the data and train the model. Depending on the volume of the data, it could take a little more time, be patient.

3

When the upload is completed, you will be automatically directed to the predict page. This is where you will input the file you want to predict. The file you want to predict should be cleaned and arranged as the file used to train the model, all the columns should be arranged the same way, however, there should be no values in the last column since that is what you want to predict. Click on Predict to run the predictions. After a successful prediction, the output file should be availble for download.
